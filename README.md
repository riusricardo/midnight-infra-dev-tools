# Midnight Infrastructure Dev Tools

This repository is a collection of useful tools for Midnight developers. The main objective is to enable developers to understand the tech stack, mainly the infrastructure.

The repository covers tools for:
- Managing the proof server
- Starting a Midnight network developer node
- Multi-node network management
- Managing the indexer
